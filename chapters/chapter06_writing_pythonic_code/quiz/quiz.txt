6. What does this code output?
    nums = [1, 2, 3, 4]
    result = [n * 2 for n in nums if n % 2 == 0]
    print(result)

Output: [4, 8]

7. What does this return?
    data = {"a": 1, "b": 2}
    print(data.get("c", 0))

Output: Means look for key “c” and if it does not exist, return 0
